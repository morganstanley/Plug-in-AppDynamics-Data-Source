## Getting started

1. Install dependencies

   ```bash
   npm install
   ```

2. Build plugin in development mode or run in watch mode

   ```bash
   npm run dev
   ```

   or

   ```bash
   npm run watch
   ```

3. Unit Testing

   ```bash
   npm run test
   ```

4. Build plugin in production mode

   ```bash
   npm run build
   ```
